  • Car and Text Recognizer

    AWS, Docker, Java

First instance scanned all the images in the S3 bucket and identified objects that are having image of an car than images with those indexes are passes through SQS queue. Instance B identifies the text on those images. The jar files were uploaded on AWS in S3 bucket. Output file is generated with the index no of car image and text written on that image. Project also uses Docker to create a container image to simplify deployment.
